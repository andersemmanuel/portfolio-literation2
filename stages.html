Stages BTS SIO

Cette page présente mes deux stages réalisés pendant le BTS SIO option SLAM : un premier stage d’initiation en première année, puis un stage d’approfondissement en deuxième année, davantage orienté développement.

Pour chaque stage, je détaille l’entreprise d’accueil, les missions confiées, les technologies utilisées, ainsi qu’une sélection de captures d’écran et de ressources associées (code, docs, liens utiles).

Stage de première année

Présentation de l’entreprise

Stage effectué au sein de Nom de l’entreprise 1, une structure spécialisée dans (à préciser : services numériques, PME locale, collectivité, etc.). L’objectif du stage était de découvrir le fonctionnement d’un service informatique et de participer à de petites évolutions applicatives.

Missions réalisées

  • Analyse de demandes simples d’évolutions ou de correctifs.
  • Participation à la mise à jour d’une application ou d’un site interne.
  • Tests et validation avec l’équipe ou le tuteur de stage.

Technologies utilisées

  • HTML / CSS pour de petites adaptations d’interface.
  • Notions de JavaScript ou d’un CMS selon le contexte.
  • Outils de suivi (tickets, mails, tableaux de suivi, etc.).

Stage de deuxième année

Présentation de l’entreprise

Missions réalisées
